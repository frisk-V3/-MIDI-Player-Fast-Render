# MIDI Player Fast Render

BPM 1000という極限のスピードでMIDIを奏でる、実験的なウェブプレイヤー。

## ⚡ Overview
このプロジェクトは、ブラウザ上で動作するMIDIプレイヤーです。
一般的な再生に加え、Tone.jsの限界に挑む**BPM 1000**でのハイスピード再生機能を搭載しています。

## 🚀 Features
- **Ultra-Fast Playback**: 通常（BPM 120）と高速（BPM 1000）の切り替え。
- **Seamless UI**: ドラッグ&ドロップによる直感的なファイル読み込み。
- **Cyberpunk Aesthetics**: ダークテーマにネオンブルーを配したUIデザイン。

## 🛠 Tech Stack
- [Tone.js](https://tonejs.github.io)
- [@tonejs/midi](https://github.com)

## 📖 How to Use
1. リポジトリをクローンまたはダウンロード。
2. `index.html` をブラウザで起動。
3. MIDIファイルをドロップして `⚡ 高速再生` をクリック。
